# Investor / Buyer README

## Product summary

Autonomous Betting Agent is an audited sports intelligence platform. It combines prediction reports, odds normalization, local multi-user profiles, proof-ledger tracking, security checks, ROI/unit reporting, and public performance dashboards.

The core positioning is not simply "AI betting picks." The stronger positioning is:

> Audited sports analytics platform with timestamped predictions, odds capture, ROI tracking, clean grading, security checks, and tamper-evident local ledgers.

## Current product strengths

- Sports prediction/report workflow.
- Odds and probability normalization.
- Audit enrichment for confidence tiers, ROI, units, and grading labels.
- Local multi-user profiles without cloud authentication.
- Proof Ledger with SHA-256 row hash chain.
- Public Results Dashboard.
- Investor Mode snapshot generation.
- Security Center for CSV/upload checks, secret redaction, and safer exports.

## Remaining commercialization steps

- Real authentication.
- Stripe subscriptions.
- Managed database.
- Broader automated final-result coverage.
- Alerts by email, Telegram, Discord, or SMS.
- External proof/notarization if needed.

## Valuation logic

The app becomes more valuable when it can prove:

1. Picks were timestamped before events.
2. Odds were captured at prediction time.
3. Results are graded cleanly.
4. ROI is positive over a meaningful sample.
5. Users are willing to pay monthly.

A $100k+ valuation is much easier to defend with hundreds or thousands of timestamped ledger rows plus positive ROI and recurring revenue.
