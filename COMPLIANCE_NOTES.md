# Compliance Notes

## Positioning

This project should be positioned as a research and analytics platform, not as guaranteed betting advice.

Suggested language:

> This software is for research, analytics, and educational use only. It does not guarantee outcomes and does not place bets for users.

## Avoid these claims

- Guaranteed wins.
- Risk-free profit.
- Certain picks.
- Insider information.
- Automated betting without user review.

## Recommended product safeguards

- Show risk disclosures in reports.
- Do not place bets directly from the app.
- Require users to follow local laws.
- Add responsible-use language.
- Keep audit results transparent.
- Separate pending/review-needed results from official performance.

## Before public launch

- Add real authentication.
- Add terms of service.
- Add privacy policy.
- Add responsible gambling resources.
- Review local betting/advertising rules in the target market.
- Use managed secret storage for API keys.
- Use HTTPS hosting and access controls.

## Current limitation

Local profiles are not secure accounts. They are only local profile selectors for demos, testing, and pre-SaaS workflows.
